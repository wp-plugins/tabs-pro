Easy Responsive Shortcode Tabs for WordPress.

== Description ==
This plugin allows you to use shortcode to display tabs post or page. you can easily create tabs using shortcode.


### Tabs Ultimate Shortcodes by http://themepoints.com

* [Tabs Live demo!&raquo;](http://themepoints.com)

<strong>Plugin Features </strong>

* Shortcode System.
* 2 Different Theme(dark, blue).
* Unlimited Theme (pro). 
* Fully responsive.
* All Browser support.
* Custom Width(pro).
* Unlimited Tabs.
* Use via shortcode.
* Support post or page.
* Enable tinymce button.
* And many more.



== Installation ==

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate plugin you will see a button on the post or page visual editor called tptabs_ultimate Shortcode. Click on it, from there you will receive the shortcode.<br />
